# CIT300-Algorithm-Analyzer
Team project for algorithm performance analysis

## Project Overview
This project implements and analyzes the performance of four different algorithms:
- Linear Search (22UG3-0504)
- Binary Search (22UG3-0504) 
- Bubble Sort (22UG3-0504)
